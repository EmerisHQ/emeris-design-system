
# Installing
1. `git clone https://github.com/EmerisHQ/emeris-design-system.git`
2. `yarn`
3. `yarn storybook`

# Stack
- Vue 3
- Typescript
- Vite (Library mode)
- Tailwind CSS
- Storybook
